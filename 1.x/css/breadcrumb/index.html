Breadcrumb

目录

  • 基本形式
  • 结合 Icon

.am-breadcrumb 面包屑导航。

基本形式

  1. 首页
  2. 分类
  3. 内容
<ol class="am-breadcrumb">
  <li><a href="#">首页</a></li>
  <li><a href="#">分类</a></li>
  <li class="am-active">内容</li>
</ol>

结合 Icon

添加图标相应 class 即可。

  1. 首页
  2. 分类
  3. 内容
<ol class="am-breadcrumb">
  <li><a href="#" class="am-icon-home">首页</a></li>
  <li><a href="#">分类</a></li>
  <li class="am-active">内容</li>
</ol>
